Rich terminal rendering guide

The CLI port is the parity floor every richer shell must clear — if a turn can't be read as story in a wrapped terminal pane, it doesn't ship. This guide pins the default port to Rich and maps each Tier S fragment, every accepts.kind, and every projected value_type to a concrete Rich renderable, with a plain-text fallback for monochrome and dumb terminals. It invents no fields: every shape below traces to VOCAB_DIGEST.md §3–§7.

Rich is the rendering target, not a dependency of the contract. The same envelope must still render through cmd2 + print() with --no-color; the fallback column is that floor-of-the-floor, and it is normative.

§0Console contract & the four gateshow the floor holds

One Console per cursor channel. Detect width from the TTY and clamp to a legible measure; detect color support and degrade in one switch. Severity is the only thing the renderer adds on top of the contract — it round-trips with KvRow.emphasis and the closed three-token ladder ok / warn / danger, never a fourth hue. 

# one console per cursor; width + color detected once, degraded once
con = Console(width=min(os.get_terminal_size().columns, 100),
              no_color=("NO_COLOR" in os.environ) or not sys.stdout.isatty())
THEME = Theme({"ok": "#6fbf8c", "warn": "#d6a35a", "danger": "#d65a4a",
               "hotkey": "reverse #d6a35a", "speaker": "bold", "dim": "#8a8472"})
gate 1
CLI floor
Every Tier S widget renders here in Rich. This guide is the reference port; richer shells degrade to it.
gate 2
Decision legibility
Every UID an open choice references via blockers.refs is printed this turn. Locked Allow cites permit + findings — both on screen.
gate 3
Time parity
Every ritual (a roll spinner, a Live reveal) is skippable to canonical-instant in one action. Honor --skip and ritual_hints.
gate 4
Input parity
Every richer modality has a typed fallback reachable the same turn: hotkey, ordinal, and raw_command all reach the same edge.

§4Metadata · grammar, affordances, info-state§7 · footer chrome

metadata.grammar feeds the raw_command parser and supplies the prompt placeholder. info_affordances[] are discoverable side-projections — rendered as a footer hint, never a state dump. info_state.dirty_kinds marks which are stale (a *); version is monotonic. 

metadata: {
  world: "credentials",
  grammar: { placeholder: "try: inspect ticket · verify id · deny" },
  info_affordances: [
    {kind:"registry",  label:"Registry",  shortcuts:["r"]},
    {kind:"ledger",    label:"Ledger",    shortcuts:["l"]},
    {kind:"directives",label:"Directives",shortcuts:["k"]},
    {kind:"help",      label:"Help",      shortcuts:["?"]} ],
  info_state: { version: 8, dirty_kinds: ["directives"] } }

Shortcuts live in their own namespace — /r or registry — and never collide with edge hotkeys. The bar is the only place hidden state is hinted; the projection itself only crosses the wire when the player asks.

§6Monochrome & narrow widthsthe floor of the floor

Monochrome & NO_COLOR

When color is gone, severity must survive in glyph + bracket + weight, never hue alone (the ladder is what carries meaning). Set Console(no_color=True) and the theme collapses to text styling Rich still honors: bold, dim, reverse, underline.

  • ok → ✓   warn → ! / [! ]   danger → ‼ / [!!]
  • hotkeys keep reverse video — readable on any monochrome TTY.
  • locked choices: [a] in brackets + [LOCKED] tag, not a faded color.
  • bars degrade to ASCII [#######-------] with the fraction always printed.
findings · --no-color
  [ok] permit_seal     Imperial
  [!!] permit_expiry   2026-03-01  expired
  [! ] origin          Kalden      review

Narrow widths · ≤ 40 cols

Below the rail's measure, the Layout split drops and everything stacks in one column. Tables fold to stacked key/value pairs; long choice text wraps under its hotkey; rules shorten to the live width. Detect once from Console.width; never hard-code 80.

  • side-by-side rail → stacked sections, same order.
  • table value_type → one key: value block per row.
  • cost previews move to their own dim line under the choice.
  • the affordance bar wraps; shortcuts never truncate.
