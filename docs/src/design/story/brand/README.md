# StoryTangl Brand References

This directory archives the reviewed brand-direction artifacts from the
design-agent UI work. The canonical source package lives at the repository root
in `brand/`; this copy keeps the design reference available in the Sphinx docs.

## v1.0

- [`v1_0/StoryTangl-Brand-Sheet.html`](v1_0/StoryTangl-Brand-Sheet.html) is the
  captured brand sheet. It is the useful artifact in this pass.
- [`v1_0/StoryTangl-brand-sheet.pdf`](v1_0/StoryTangl-brand-sheet.pdf) is the
  print-to-PDF visual reference for the same sheet.
- [`v1_0/USAGE.md`](v1_0/USAGE.md) records the implied rules for wordmark,
  palette, type, and tone.
- `v1_0/assets/` contains the reviewed source SVGs, CSS tokens, and CLI splash
  text.

Hidden screenshot/render-check files from the handoff package are intentionally
not archived here.
